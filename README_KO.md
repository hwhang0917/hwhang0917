## 안녕하세요!

풀스택 웹 개발자 황희상입니다!

영어 이름으로는 Mark를 사용합니다! (아니면 웹에서는 `달리는냉장고`도 사용해요 😜)

웹 개발자면서 CLI 개발 환경을 매우 좋아합니다 ❤️❤️

최근에는 [wezterm](https://wezfurlong.org/wezterm/)을 메인 터미널로 사용 중입니다!

[여기]()에서 저의 wezterm 설정을 볼 수 있습니다.
