## 안녕하세요!

풀스택 웹 개발자 황희상입니다!

영어 이름으로는 Mark를 사용합니다! (아니면 웹에서는 `달리는냉장고`도 사용해요 😜)

저는 CLI 툴을 좋아하고, CLI 툴을 사용하여 제 워크플로우를 자동화하는 것을 즐깁니다.

[이곳](https://github.com/hwhang0917/dotfiles)에서 제 설정 파일들을 확인하실 수 있습니다!
